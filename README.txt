LAPSTORE — Orang 2 - Catalog Detail Search

Isi ZIP:
- Base project Vite React
- Shared data dan shared components
- File halaman khusus ORANG2: src/pages/orang2_catalog_detail_search.jsx
- App.jsx sudah disetel agar cuma membuka bagian ORANG2 + halaman base

Cara menjalankan:
1. Extract ZIP ini
2. Buka folder LapStore_orang2
3. Jalankan:
   npm install
   npm run dev

Bagian interface:
Product Catalog, Product Detail, dan Search Result.

Catatan:
Kalau klik menu halaman orang lain, aplikasi tidak error. Akan muncul halaman info bahwa halaman itu tidak ikut ZIP ini.
