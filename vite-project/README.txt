LAPSTORE — Orang 3 - Category Cart Checkout

Isi ZIP:
- Base project Vite React
- Shared data dan shared components
- File halaman khusus ORANG3: src/pages/orang3_category_cart_checkout.jsx
- App.jsx sudah disetel agar cuma membuka bagian ORANG3 + halaman base

Cara menjalankan:
1. Extract ZIP ini
2. Buka folder LapStore_orang3
3. Jalankan:
   npm install
   npm run dev

Bagian interface:
Category Page, Shopping Cart, dan Checkout Page.

Catatan:
Kalau klik menu halaman orang lain, aplikasi tidak error. Akan muncul halaman info bahwa halaman itu tidak ikut ZIP ini.
