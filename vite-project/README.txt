LAPSTORE — Orang 4 - Success Orders Wishlist

Isi ZIP:
- Base project Vite React
- Shared data dan shared components
- File halaman khusus ORANG4: src/pages/orang4_success_orders_wishlist.jsx
- App.jsx sudah disetel agar cuma membuka bagian ORANG4 + halaman base

Cara menjalankan:
1. Extract ZIP ini
2. Buka folder LapStore_orang4
3. Jalankan:
   npm install
   npm run dev

Bagian interface:
Payment Success, Order History, dan Wishlist Page.

Catatan:
Kalau klik menu halaman orang lain, aplikasi tidak error.
